🎯 Core Rules
1

Sequential Learning

Questions are asked in order — from the first to the last. No skipping allowed.

2

Wrong Answer = Restart

If you get any question wrong, the correct answer is shown and you restart that category from Question 1.

3

Complete to Unlock

A category is only marked complete when you answer ALL questions correctly in one unbroken run.

4

Checkpoints for Large Categories

Categories with more than 50 questions (like Tables 1–20) are split into Checkpoints of 50. Complete each checkpoint independently.

5

Infinite Mode = Unlocked Content Only

Infinite Mode draws random questions exclusively from categories you've already completed. Finish more to expand the pool.

💾 Save & Resume Progress

Your progress is NOT saved automatically.

Mug Up Master runs entirely in your browser — there's no account or cloud storage. But you can export your progress as a JSON file at any time and re-import it later to continue exactly where you left off — even on a different device.

How it works:
1. Click Save / Load in the top-right nav bar.
2. Press Export Progress — a mug-up-progress.json file is downloaded.
3. Next time, open the app, click Save / Load again, then Import Save File.
4. Your completed categories and checkpoints are instantly restored.
💡 Tips for Success
⌨️

Use Enter Key

Press Enter to submit your answer — no need to click the button.

📖

Read the Hint

Each question has a small hint below it. Read it carefully when stuck.

🎯

Case Insensitive

Answers are not case-sensitive. "Ganga" and "ganga" are both accepted.

🌙

Dark / Light Mode

Toggle between dark and light themes using the button in the top-right corner.

🔁

Navigation is Locked

Once inside a module, only the Home button takes you back — preventing accidental exits mid-session.

📦

Checkpoint System

Big categories are split into 50-question checkpoints. Each checkpoint counts separately so you never lose too much progress.
